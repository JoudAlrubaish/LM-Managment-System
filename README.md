# Language Model Management System using OOP

A Python project that applies Object-Oriented Programming concepts by managing simulated language models.

The project includes two models:

- GPTModel
- LlamaModel

Note : no real API or API key is used

## OOP Concepts

- Classes and objects
- Constructors
- Instance and class attributes
- Class methods
- Static methods
- Encapsulation
- Properties
- Inheritance
- Method overriding
- Polymorphism
- Abstraction
- Composition
- Aggregation
- Multiple inheritance
- Magic methods
- Custom exceptions
